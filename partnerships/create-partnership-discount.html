1. Log in to braintreegateway.com with PayPal credentials
  2. Navigate to Subscriptions in the header
  3. Select “Add Ons/ Discounts” in the sub-header
  4. Scroll to bottom and click “New Discount”
  5. Enter a ID that is unique to the partner. The ID pattern is generally <partner name>-<subscription period>-<percent discount> eg “SUNRISE-QTR-50” is 50% discount for sunrise employees for quarterly subscriptions
  6. Enter a name for it that describes who the discount is for. Customers will see this name. 
  7. Calculate and enter the discount amount for the membership term you are working on.
  8. Each level of membership will need its own discount.
  9. Create a new page in this wiki for the partner that combines the correct membership option with the discount. See “Sunrise Labs” example for these URLs. The membership option should be the same between partners, only the discount ID changes.
