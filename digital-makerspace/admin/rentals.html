Rental Management

 

Creating a New Rental:

  1. Login to members.manchestermakerspace.org as an admin user.
  2. Using the “hamburger” menu in the upper right hand corner, navigate to “Rentals”.
  3. Click the blue button in the upper left hand corner labelled “Create New Rental”.
  4. Fill out the rental information. Number refers to a unique identifier for the rental. Description is usually used for listing the size and price of the rental.
  5. Submit - If accepted, the rental has been created
  6. The next time the member logs in, they will be asked to sign the Rental Agreement

Renewing a Rental:

  1. Rentals can be renewed from the Rentals list or from a member’s profile. The process is the same
  2. Login to the service as an admin user.
  3. With one of the following instructions, view a list of rentals.
  4. Using the “hamburger” menu in the upper right hand corner, navigate to “Rentals” .
  5. Navigate to a member’s profile that owns the rental you’re looking for and select the “Rentals” tab.
  6. Select the checkbox for the row of the rental you want to renew and click the “Renew Rental” button at the top of the table.
  7. Select the term that the member paid for from the dropdown and submit the form.
  8. When submitted, the rental is renewed, sending a Slack message to #members_relations.

Deleting a Rental:

  1. If a member has decided they no longer want the rental, you may delete the rental in the Digital Makerspace in order to stop notifications
  2. Login to members.manchestermakerspace.org as an admin user.
  3. Using the “hamburger” menu in the upper right hand corner, navigate to “Rentals”.
  4. Select the checkbox for the rental you would like to delete
  5. Click the red Delete Rental box and accept the confirmation prompt.
  6. When submitted, the rental is deleted and removed from the list, this will also delete any invoices associated with  the rental.
